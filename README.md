EngeneWeather adalah aplikasi dashboard pemantauan cuaca berbasis web yang dirancang untuk menyajikan informasi meteorologi secara real-time dengan antarmuka modern bergaya Glassmorphism. Aplikasi ini mengintegrasikan data akurat dari OpenWeatherMap API menggunakan PHP sebagai backend proxy untuk menjamin keamanan akses data, sementara sisi antarmuka dibangun secara responsif menggunakan HTML, JavaScript, dan Tailwind CSS. Fitur unggulan EngeneWeather meliputi visualisasi indikator lingkungan (suhu, kelembapan, angin) melalui grafik interaktif ApexCharts, prediksi cuaca presisi untuk 5 hari ke depan, serta dukungan personalisasi pengguna melalui fitur Dark/Light Mode dan penyimpanan kota favorit berbasis LocalStorage. Dengan arsitektur yang efisien tanpa membebani penyimpanan database konvensional, EngeneWeather menawarkan pengalaman pemantauan cuaca yang informatif, estetis, dan cepat.
